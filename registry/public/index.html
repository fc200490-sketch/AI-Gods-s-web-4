The new directory for the agentic web.

Every service publishes a machine-readable manifest. Every agent discovers by intent, not by keyword. Reputation is earned in public — uptime, ratings, calls. The infrastructure the Internet forgot to build.

Editorial note This is a working prototype of a centralized registry for agent-callable services. Manifests carry JSON-Schema I/O contracts and a pricing slot (x402 wired next). Discovery is capability-first. The protocol is HTTP + JSON, by design.
